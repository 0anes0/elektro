## 🚀 Hakkında
**Elektro**, elektrik-elektronik mühendisliği hesaplamalarını, devre analizlerini ve temel formülasyonları doğrudan Linux/macOS terminali üzerinden yapmanızı sağlayan hızlı ve pratik bir komut satırı (CLI) aracıdır.

## ⚙️ Gereksinimler
Kuruluma geçmeden önce sisteminizde aşağıdakilerin bulunduğundan emin olun:
* **Python 3.8+**
* **Git**
* **pipx** (Modern Python paketlerini sistem ortamını bozmadan izole bir şekilde global olarak kurmak için)
  * *Ubuntu/Debian için:* `sudo apt install pipx`
  * *Arch Linux için:* `sudo pacman -S pipx`

## 🛠️ Kurulum

Uygulamayı sisteminize global bir komut olarak kurmak için aşağıdaki adımları sırasıyla terminalinize yapıştırın.

**1. Repoyu bilgisayarınıza klonlayın ve dizine girin:**
```bash
git clone [https://github.com/0anes0/elektro.git](https://github.com/0anes0/elektro.git)
cd elektro
```

**2. Uygulamayı sisteme izole ve global bir şekilde kurun:**
```bash
pipx install .

#pipx kurulu değilse kurun.

```
